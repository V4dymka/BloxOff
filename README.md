# BloxOff

**BloxOff** is a small 3D game built with Three.js where you control a cube and jump across platforms.  
The game works offline and supports both keyboard controls and mobile touch controls (joystick and jump button).

## How to Run
1. Clone the repository or download the ZIP archive.
2. Open `index.html` in your browser.
3. For best stability, run the game through a local server (for example: `npx http-server`).

## Controls
- **PC**: WASD or Arrow keys to move, Space to jump.
- **Mobile**: Virtual joystick + Jump button.

## Features
- Fullscreen support.
- Platform collisions.
- Mobile controls are automatically hidden on PC and iPad.
